📊 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

ПУБЛИЧНЫЕ МАРШРУТЫ (без аутентификации):

  POST /api/appointment/request-code
    → Запросить код подтверждения для записи
    Body: { fullName, phone, email, vehicleType, vehicleModel, appointmentDate, appointmentTime, ... }

  POST /api/appointment/verify
    → Подтвердить запись по коду
    Body: { email, code }

  POST /api/appointment/resend-code
    → Повторно отправить код подтверждения
    Body: { email }

  POST /api/contact
    → Отправить контактное сообщение
    Body: { fullName, email, phone, subject, message }


ЗАЩИЩЕННЫЕ МАРШРУТЫ (требуется JWT токен в Authorization header):

  ╔ АДМИНИСТРАТОР
  ║
  ╠─ POST /api/auth/login
  ║    → Вход администратора
  ║    Body: { username, password }
  ║    Returns: { token, admin: { id, username, email, role } }
  ║
  ╠─ GET /api/auth/verify
  ║    → Проверить валидность токена
  ║    Headers: Authorization: Bearer <token>
  ║
  ╠─ POST /api/auth/create-admin
  ║    → Создать нового администратора
  ║    Body: { username, email, password, role }
  ║
  ║
  ╠ УПРАВЛЕНИЕ ЗАПИСЯМИ
  ║
  ╠─ GET /api/admin/appointments
  ║    → Получить все записи на техосмотр
  ║
  ╠─ GET /api/admin/appointments/:id
  ║    → Получить запись по ID
  ║
  ╠─ PUT /api/admin/appointments/:id/cancel
  ║    → Обновить статус записи
  ║    Body: { status: "pending|confirmed|completed|cancelled" }
  ║
  ║
  ╠ УПРАВЛЕНИЕ СООБЩЕНИЯМИ
  ║
  ╠─ GET /api/admin/messages
  ║    → Получить все контактные сообщения
  ║
  ╠─ GET /api/admin/messages/:id
  ║    → Получить сообщение по ID
  ║
  ╠─ PUT /api/admin/messages/:id
  ║    → Обновить статус сообщения
  ║    Body: { status: "new|read|answered" }
  ║
  ╠─ DELETE /api/admin/messages/:id
       → Удалить сообщение
  ║
  ╚


⚙️  НАСТРОЙКА ОКРУЖЕНИЯ (.env)
═══════════════════════════════════════════════════════════════════════════════

Необходимые переменные в файле .env:

  # Сервер
  PORT=3000
  NODE_ENV=production

  # База данных
  DATABASE_URL=postgresql://avtokond_user:password@avtokond_db:5432/avtokond_prod

  # JWT (используйте сильный, случайный ключ)
  JWT_SECRET=your-super-secret-key-here-minimum-32-characters

  # Email (Yandex)
  SMTP_HOST=smtp.yandex.com
  SMTP_PORT=587
  SMTP_USER=your-email@yandex.com
  SMTP_PASS=your-app-password
  ADMIN_EMAIL=admin@avtokond-ugra.ru
