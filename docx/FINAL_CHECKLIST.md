### Рекомендации по развертыванию

1. **Обновить пароль администратора:**
   ```
   Логин > Новый пароль (сильный, 12+ символов)
   ```

2. **Проверить переменные окружения в .env:**
   ```
   JWT_SECRET - надежный ключ (32+ символа)
   SMTP_* - правильные параметры Yandex
   ADMIN_EMAIL - корректный адрес
   ```

3. **Проверить логи при запуске:**
   ```
   docker-compose logs backend
   ```

4. **Протестировать основные функции:**
   - ✓ Залогиниться в админ-панель
   - ✓ Просмотреть записи/сообщения
   - ✓ Отправить тестовую форму
   - ✓ Проверить приход кода на email

5. **Настроить резервные копии БД:**
   ```
   docker exec avtokond_db pg_dump -U user dbname > backup.sql
   ```

6. **Мониторить логи в продакшене:**
   ```
   docker logs -f avtokond_backend
   ```
