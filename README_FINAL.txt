📊 API ENDPOINTS (для разработчиков)
═══════════════════════════════════════════════════════════════════════════════

ЗАПИСИ НА ТЕХОСМОТР:
  POST   /api/appointment/request-code  - запросить код
  POST   /api/appointment/verify        - подтвердить запись
  POST   /api/appointment/resend-code   - отправить код заново
  GET    /api/admin/appointments       - получить все (JWT)
  GET    /api/admin/appointments/:id   - получить одну (JWT)
  PUT    /api/admin/appointments/:id   - обновить статус (JWT)

КОНТАКТНЫЕ СООБЩЕНИЯ:
  POST   /api/contact                  - отправить сообщение
  GET    /api/admin/messages           - получить все (JWT)
  GET    /api/admin/messages/:id       - получить одно (JWT)
  PUT    /api/admin/messages/:id       - обновить статус (JWT)
  DELETE /api/admin/messages/:id       - удалить (JWT)

АУТЕНТИФИКАЦИЯ:
  POST   /api/auth/login               - вход администратора
  GET    /api/auth/verify              - проверить токен (JWT)
  POST   /api/auth/create-admin        - создать администратора (JWT)

ПРОБЛЕМА: Админ-панель не загружается
РЕШЕНИЕ:
  ✓ Обновите страницу (F5)
  ✓ Очистите кэш браузера (Ctrl+Shift+Delete)
  ✓ Проверьте консоль браузера (F12 > Console) на ошибки
  ✓ Убедитесь, что бекенд запущен (docker-compose ps)

ПРОБЛЕМА: Не удается залогиниться
РЕШЕНИЕ:
  ✓ Проверьте, что вводите пароль правильно (учитывается регистр)
  ✓ Убедитесь, что .env файл содержит JWT_SECRET
  ✓ Проверьте логи бекенда (docker-compose logs backend)

ПРОБЛЕМА: Email не приходят
РЕШЕНИЕ:
  ✓ Проверьте папку спам в почте
  ✓ Убедитесь, что SMTP параметры правильные в .env
  ✓ Проверьте что ADMIN_EMAIL указан правильно
  ✓ Проверьте логи бекенда (docker-compose logs backend)

ПРОБЛЕМА: База данных не подключается
РЕШЕНИЕ:
  ✓ Проверьте DATABASE_URL в .env файле
  ✓ Убедитесь, что PostgreSQL контейнер запущен (docker ps)
  ✓ Проверьте логи БД (docker-compose logs db)
